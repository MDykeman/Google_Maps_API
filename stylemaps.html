How To: Style Google Maps

Intro:

In these instructions you will be able to utilize the Google Maps API, and style the map to suit your needs. You should be familiar with basic code editors, CSS, HTML and Javascript. This isn't going to be hard!

Set up a basic page and get Google Maps:

Google makes it really easy to get a map on your page quickly. They provide the HTML, some styling and Javascript right out of the box. Go here to grab the code(click on the 'Javascript + HTML' button) and paste it into your editor.

If this works, you will see Google Maps taking up your entire browser window, congratulations.

Targeting your "map-canvas" DIV in CSS:

Create a CSS file and link it to your page. Once you have your CSS file linked correctly you will be able to start manipulating style attributes. Start with the width of your map by typing:
#map-canvas {
width: 80vw;
}.

If the width of your map DIV changes, you have linked the CSS correctly; you can now use other CSS attributes.

Modify the style of your map:

Now the fun part. Within your SCRIPT tag that holds your map, enclose your entire maps code within a function, a function that will be called when your window loads in your browser. Like this:
window.onload = function(){
YOUR MAPS CODE HERE
}

Once your map is inside an onload function, you will need to add the code that will style your map. Above your maps code, but still inside the onload function, place this code: var styles = [
{
featureType: 'YOUR CHOSEN FEATURE HERE',
elementType: 'YOUR PROPERTY HERE',
stylers: [
{ YOUR CHOSEN FEATURE HERE}
]
}
];

The 'styles' variable holds all your styles that you want applied to your map.

There are three different areas to specify: the 'featureType', the 'elementType' and the 'stylers'. The 'featureType' selects the the physical objects in your map such as water, roads, and landscapes(to name a few). The 'elementType' allows you to target properties of the chosen 'featureType', such as the fill, stroke, and shape(to name a few).

Before you start adding styles, you have to tell the map that you will be styling it using your specified styles.
