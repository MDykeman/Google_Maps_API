How To: Style Google Maps

Intro:

In these instructions you will be able to utilize the Google Maps API, and style the map to suit your needs. You should be familiar with basic code editors, CSS, HTML and Javascript. This isn't going to be hard!

Targeting your "map-canvas" DIV in CSS:

Create a CSS file and link it to your page. Once you have your CSS file linked correctly you will be able to start manipulating style attributes. Start with the width of your map by typing this in your CSS file:

#map-canvas {
width: 80vw;
}.

If the width of your map DIV changes, you have linked the CSS correctly; you can now use other CSS attributes.

Modify the style of your map:

Wrap your map code in a window.onload function like so:

window.onload = function(){
YOUR MAP CODE HERE
}

Place this code below your map's code, but still within the onload function:

var styles = [
{
featureType: 'water',
elementType: 'geometry.fill',
stylers: [
{ color: '#adc9b8' }
]
}
];

The 'featureType' selects parts of the map like water, roads and landscapes. The 'elementType' selects what you want to change, such as the fill or shape. The 'stylers' allows you to specify what you changing about it, such as the color or transparency.

Connect your styles to your map by placing this line of code at the bottom of your onload function:

map.setOptions( { styles: styles } );

The 'styles' portion will be the variable that you called your styles.
